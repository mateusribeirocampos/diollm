## Installation

### Clone repository

```bash
git clone https://github.com/mateusribeirocampos/diollm.git
cd diollm
```

### Create virtual environment

```bash
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
.\.venv\Scripts\activate  # Windows
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Launch Jupyter

```bash
jupyter notebook
```
